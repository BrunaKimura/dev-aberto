Avaliação

Como vocês podem ver na aula de apresentação, essa disciplina permite uma liberdade enorme e a maneira que isto está organizado é via gameficação. Cada participante cria um perfil no repositório e ganha skills (e XP) ao entregar as possíveis atividades. Toda entrega é feita via PR para o repositório da disciplina.

A composição da nota é um misto da quantidade total de XP recebida por enviar tarefas, a quantidade de XP obtida por membros de seu grupo e alguns critérios de barreira. A tabela abaixo resume essas condições:

Conceito Pontuação Critérios extras
I ---- Não fez alguma das atividades de tutorial
D ---- Completou todas atividades de aula, mas não completou as atividades para o C
C 60XP Obteve skill dos três tipos:
  • Contribuição de código
  • Tradução aceita!
  • Atividade de comunidade
    		•
    B 100XP
  • Contribuição de código aceita
  • Participação em evento de tecnologia
    		•
    A 200XP

    Durante o semestre as pontuações da tabela acima podem passar por revisões para baixo, mas nunca para cima.

    Skills disponíveis

    Cada skill se traduz em evidências para um dos objetivos de aprendizagem da disciplina. O conjunto de todas as skills feitas no semestre indica a proficiência dos estudantes em cada um dos objetivos.

    Tutorial

    Nesta parte inicial são feitas atividades preparatórias que apoiam o desenvolvimento das skills de Código, Documentação e Tradução e Comunidade.

    ID Descrição
    1 Primeiros passos Se adicionou no repositório da disciplina. 2XP
    2 Expectativas Completou atividade de expectativas da disciplina. 2XP
    3 Explorando o entorno de um projeto Pesquisou sobre a história e a comunidade de pessoas e serviços ao redor de um projeto. 2XP
    6 Projeto não documentado Conseguiu rodar o sistema flask de desafios. 2XP
    7 Tradução básica Usou ferramentas de tradução para traduzir e localizar um pequeno programa em linha de comando. 2XP
    8 Projeto Profissional Reformou um código mal feito e criou uma página web para ele. 5XP
    9 Pacote Python Criou um pacote python instalável via pip. 2XP
    10 Projeto DIY Contribuiu com um projeto educacional livre do Insper 5XP

    Código

    Toda atividade que resulte em uma contribuição de código com projetos de software aberto é aceita na disciplina.

    ID Descrição
    4 BUG simples Enviou um PR corrigindo um bug simples. 5XP
    5 BUG complexo Enviou um PR corrigindo um bug complexo. 10XP
    20 Bug Report Reportou um bug em um repositório e outro usuário conseguiu reproduzí-lo. 1XP
    21 Feature Request Fez um pedido de nova funcionalidade a um software. 1XP
    22 Contribuição aceita! Um PR não trivial foi aceito por um projeto externo. 5XP
    23 Issue indireta Conversas em uma issue ou PR fizeram outros usuários abrirem issues relatando novos bugs/melhorias. 2XP
    32 Blogueiro Criou tutorial de um software ou biblioteca em um blog (ou sites como Medium) 5XP
    33 youtuber Criou video tutorial de um software ou biblioteca. 5XP

    Tradução e documentação

    Contribuições para documentações e traduções de softwares e materiais de ensino ligados a tecnologia e software livres são aceitas neste eixo.

    ID Descrição
    30 Tradução aceita! Fez uma tradução que foi aceita em um software aberto. 5XP
    31 Novas línguas Adicionou suporte a traduções em um software que não era traduzível. 10XP

    Comunidade

    Um projeto de software também é definido pelas interações de todas as pessoas ao seu redor: usuários, desenvolvedores e tradutores. Neste quesito entram atividades de interação com pessoas em um âmbito menos acadêmico e mais profissional, como:

    1. participação em eventos
    2. palestras
    3. contribuições em projetos internos do Insper
    ID Descrição
    40 Participação em Evento Participou de um evento da área de Tecnologia. 7XP
    41 Metacontribuição Fez uma contribuição simples no repositório da matéria. 2XP
    42 Palestrante Fez uma palestra sobre bibliotecas ou softwares livres. 10XP
    43 Helpdesk Teve uma resposta aceita em sites como StackOverflow ou em listas de email 4XP
